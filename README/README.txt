	Dựa trên nhu cầu lập đơn hàng và tính toán công nợ hàng tháng và xuất ra file Excel, ứng dụng làm ra để hỗ trợ công việc trở lên hiệu quả và năng suất hơn
	Dữ liệu gồm thông tin về các Mặt hàng và các Đơn hàng được lưu vào cơ sở dữ liệu SQLite trong máy.
	Xuất file Excel gồm các đơn hàng của từng ngày, từng công ty.
	Tính toán tổng tiền của tháng và xuất thành file Excel.
	

	Những thứ học được trong dự án:
	-Xây dựng bố cục, layout cơ bản
	-Thao tác sử dụng với ListView, ExpandableListView
	-Truy xuất dữ liệu từ cơ sở dữ liệu SQLite
	-Sử dụng thư viện hỗ trợ Apache Poi để xuất file Excel
	-Truyền và nhận dữ liệu giữa các Activity
	-Xây dựng mô hình MVP, MVC
	-Thiết kế class và giao tiếp giữa các class
